**Assignment 2: Trees and Graphs Implementation**

1. **Assignment Objectives**

This assignment is designed to assess your ability to:

* Understand and apply Tree and Graph data structures  
* Implement Binary Search Tree operations  
* Implement Graph traversal and graph algorithms  
* Solve problems using appropriate data structures  
* Demonstrate good programming and algorithmic thinking

**Part A: Binary Search Tree (50 Marks)**  
You are required to implement a **Binary Search Tree (BST)** using Java (or any language specified by lecturer).

1) **Task A1: BST Implementation (20 Marks)**  
   Create a BST class that supports the following operations:  
* insert(value)  
* search(value)  
* delete(value)  
* findMin()  
* findMax()

2) **Task A2: Tree Traversal (15 Marks)**

Implement the following traversal methods:

* Inorder Traversal  
* Preorder Traversal  
* Postorder Traversal

3) ## **Task A3: Tree Properties (10 Marks)**

Implement functions to calculate:

* Height of the tree  
* Total number of nodes  
* Number of leaf nodes

4) ## **Task A4: BST Demonstration Program (5 Marks)**

Create a menu-driven program that allows user to:

1. Insert value  
2. Delete value  
3. Search value  
4. Display traversals  
5. Display tree properties  
6. Exit

# **Part B: Graph Implementation (50 Marks)**

You are required to implement a Graph using adjacency list or adjacency matrix.

1) ## **Task B1: Graph Representation (10 Marks)**

Implement a Graph class that can:

* Add vertex  
* Add edge  
* Display graph

2) ## **Task B2: Graph Traversal (15 Marks)**

Implement:

* Breadth-First Search (BFS)  
* Depth-First Search (DFS)

3) ## **Task B3: Shortest Path Algorithm (15 Marks)**

Implement:

* Dijkstra’s Algorithm

The program should:

* Allow user to choose source node  
* Display shortest path to all other nodes  
4) **Task B4: Minimum Spanning Tree (10 Marks)**

Implement one of the following:

* Prim’s Algorithm  
  or  
* Kruskal’s Algorithm

Display:

* The edges selected  
* Total cost of spanning tree

**Report Requirements**

You must submit a short report containing:

1. Introduction  
2. Explanation of BST Implementation  
3. Explanation of Graph Implementation  
4. Screenshots of program output  
5. Conclusion

All submissions must be made through FlexLearn and OAS within the timeline given.
